# Edge Store Submission Bundle

This folder contains draft materials for Microsoft Edge Add-ons submission.

## Included drafts

- `privacy-policy.md`: public-facing privacy policy draft. Host this on a public webpage and use that URL in Partner Center.
- `support.md`: public-facing support page draft. Host this on a public webpage or convert it into a support/help page.
- `store-listing.md`: paste-ready store listing copy, metadata recommendations, and search terms.
- `certification-notes.md`: paste-ready notes for certification, including placeholders for any required test credentials.
- `visual-assets.md`: required sizes plus a screenshot shot list and tile copy.

## Generated assets

- Use [icon-300.png](../icons/icon-300.png) as the store logo draft.
- Use the generated promotional tile drafts in [assets](./assets).
- When this repo is published through GitHub Pages from `/docs`, the public URLs will typically be:
  - Home: `https://<github-username>.github.io/<repo>/`
  - Privacy: `https://<github-username>.github.io/<repo>/privacy/`
  - Support: `https://<github-username>.github.io/<repo>/support/`

## Replace before submit

- Replace placeholder URLs and email addresses.
- Add real test credentials, if certification requires Microsoft 365 Copilot access.
- Capture real screenshots from the packaged extension in Edge.
