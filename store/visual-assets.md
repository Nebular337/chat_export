# Visual Assets Draft

The following sizes are based on the current Microsoft Edge publishing docs.

## Required or recommended assets

- Extension logo: aspect ratio `1:1`, recommended `300 x 300`, minimum `128 x 128`
- Small promotional tile: `440 x 280`
- Large promotional tile: `1400 x 560`
- Screenshots: up to `6`, each either `640 x 480` or `1280 x 800`

## Screenshot shot list

Capture these from the packaged extension running in Edge:

1. A Microsoft 365 Copilot Chat conversation open on `m365.cloud.microsoft/chat` with the extension available in the toolbar.
2. The export preview tab showing the formatted transcript.
3. A transcript section that includes lists, links, or table content.
4. The header area with the `Print to PDF` action visible.

## Screenshot guidance

- Use real data only if you are allowed to share it. Otherwise use a safe test conversation.
- Keep the page zoom at a readable level.
- Avoid browser UI clutter that distracts from the extension's function.
- Make sure the screenshots match the exact shipped build.
