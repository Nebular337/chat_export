Export Microsoft 365 Copilot Chat into a cleaner print preview.

Copilot Chat Exporter turns the conversation already open on m365.cloud.microsoft/chat into a structured transcript preview for printing or saving as PDF.

Get support Read privacy policy

Not affiliated with Microsoft. Supports Microsoft Edge and Microsoft 365 Copilot Chat on m365.cloud.microsoft/chat.

What it does

  • Exports the current Microsoft 365 Copilot Chat conversation.
  • Opens a dedicated, print-friendly transcript preview in a new tab.
  • Preserves useful structure such as headings, lists, links, citations, tables, and images where possible.
  • Lets the user print or save the conversation through Edge’s normal print flow.

What it does not do

  • It is not an official Microsoft product.
  • It does not support copilot.microsoft.com.
  • It does not upload exported transcript contents to a publisher-controlled server during normal use.
